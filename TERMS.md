# Public Demo Terms

This repository and hosted demo are a public proof of concept. They are not a production marketplace, payment rail, supplier network, or external execution service.

## Allowed Use

- Use public or synthetic inputs only.
- Use the API to inspect route decisions, readiness status, OpenAPI shape, and demo outcome recording.
- Use local verification commands to inspect the zero-spend, zero-external-execution contract.
- Share non-sensitive feedback through GitHub Issues or the metadata-only feedback endpoints.

## Prohibited Use

- Do not submit private, work, company, customer, credential, local-file, internal, or regulated data.
- Do not use the demo for production, customer, medical, legal, financial, trading, security exploitation, or emergency workflows.
- Do not attempt to bypass rate limits, input filters, disabled dispatch, or public-demo boundaries.
- Do not treat a route decision as permission to perform public posting, account mutation, paid execution, supplier execution, or work on private data.

## Current Limits

- No warranties, uptime promises, durability promises, or payment terms are offered.
- Outcome records in hosted-demo mode are process memory only and may disappear at any time.
- Dispatch is disabled in V0. The service does not call suppliers, LLM APIs, paid APIs, public writes, or account-mutating tools.
- Public beta access, if offered, is for feedback only and may be paused or changed at any time.
