# Example Results

These are the expected first-user outcomes for the guided demo.

## Use Public Path

Expected result:

- Recommended path is a public or owned local proof path.
- Cost is 0 USD.
- External writes are blocked.
- Proof packet can be generated locally.
- Human approval is required before any external submission.

Evaluator signal:

- The route should feel like a practical first step for an agent that received a broad outcome.

## Use Or Build

Expected result:

- BEAN compares owned-agent, public-path, build-decision, and blocked supplier options.
- The selected path explains quality, speed, cost, risk, and proofability.
- The build-new option remains visible as an alternative, not a hidden product claim.

Evaluator signal:

- The route should clarify whether the user should reuse an existing path or invest in a new agent.

## Block Risky Work

Expected result:

- Paid API calls and public posting are blocked.
- No dispatch happens.
- Cost remains 0 USD.
- The memo explains what a human would need to approve before any external action.

Evaluator signal:

- The route should make refusal useful, not just say no.
